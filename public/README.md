# پوشه Public

این پوشه برای فایل‌های استاتیک استفاده می‌شود که مستقیماً در root سایت قرار می‌گیرند.

## فایل‌های موجود:

- `vite.svg` - لوگوی Vite
- `favicon.ico` - آیکون سایت

## استفاده:

فایل‌های موجود در این پوشه را می‌توانید مستقیماً با مسیر `/filename` در کد استفاده کنید.

مثال:
```tsx
<img src="/vite.svg" alt="Logo" />
```

## نکته:

Vite به صورت خودکار فایل‌های این پوشه را در root سایت قرار می‌دهد و نیازی به import کردن آن‌ها نیست.
